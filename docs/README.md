# Bloops Documentation

This directory houses the planning, design, and workflow assets for the Bloops MVP.
Use the sections below to locate the artifact you need.

## Directory Overview

- **`design/`** – Visual identity resources, including `design-tokens.json` for colors, type, and spacing
- **`process/`** – Developer onboarding and delivery guides such as `quick_onboarding.md` and `build-checklist.md`
- **`product/`** – Product planning material like `mvp-scope.md`, `milestone-plan.md`, and `user-stories.md`
- **`reports/`** – Historical test reports, QA audits, and implementation summaries
- **`templates/`** – Reusable GitHub issue templates (`ISSUES.md`)
- **`images/`** – Visual assets and mockups

## Recommended Reading Order

1. **[process/quick_onboarding.md](process/quick_onboarding.md)** for the development workflow and tooling setup
2. **[product/mvp-scope.md](product/mvp-scope.md)** to understand the feature boundaries of the release
3. **[product/user-stories.md](product/user-stories.md)** and **[product/milestone-plan.md](product/milestone-plan.md)** for implementation context
4. **[design/design-tokens.json](design/design-tokens.json)** when working on UI components
5. **[reports/README.md](reports/README.md)** for quality assurance and testing documentation
